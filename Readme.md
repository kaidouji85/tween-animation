# Tweenライブラリサンプル

## 動かし方
1. Readme.mdがある場所で、以下コマンドを実行する

```
npm install
npm start
```

2. 実行したいサンプルごとに、以下URLをブラウザで開く

|  サンプル名               | URL                                     | 
| ----------------------- | --------------------------------------- |
|  01 Tween               | localhost:8080/01-tween                 |
|  02 Web Animation API   | localhost:8080/02-web-animation-api     |
|  03 アンチパターン         | localhost:8080/03-anti-pattern          |
